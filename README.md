# Student Grade Tracker (Python Assignment - Part 1)

## Overview
This project is a command-line based Student Grade Tracker built using core Python concepts. It performs data cleaning, marks analysis, class performance evaluation, and string manipulation.



## Tasks Implemented

### Task 1 — Data Parsing & Profile Cleaning
- Cleaned raw student data (names, roll numbers, marks)
- Converted marks from string to list
- Validated student names
- Displayed formatted student profiles



### Task 2 — Marks Analysis
- Assigned grades based on marks using loops and conditionals
- Calculated total and average marks
- Identified highest and lowest scoring subjects
- Implemented a marks-entry system using a while loop
- Handled invalid inputs safely


### Task 3 — Class Performance Summary
- Calculated average marks for each student
- Assigned Pass/Fail status
- Displayed a formatted class report
- Counted number of passing and failing students
- Identified class topper
- Computed overall class average


### Task 4 — String Manipulation Utility
- Cleaned and formatted a given essay
- Converted text to Title Case
- Counted occurrences of a word
- Replaced words with formatted output
- Split essay into sentences
- Displayed numbered sentences
